SUDOKU 
Sudoku is a logic-based puzzle game of number-placement. In classic sudoku, the objective is to fill a 9×9 grid with digits so that each column, each row, and each of the nine 3×3 subgrids that compose the grid contains all of the digits from 1 to 9 and non-repeated.

MAZE 
You have to decide how to escape the digital maze. The maze is an 9-by-18 grid with each position (cell) containing a positive integer. You start in the top-left corner at (1, 1) and exit from the bottom-right corner at (9, 18). If you are in a cell containing the value X, then you can jump to any cell (a, b) satisfying a × b = X. For example, if you are in a cell containing a 6, you can jump to cell (2, 3). Note that from a cell containing a 6, there are up to four cells you can jump to: (2, 3), (3, 2), (1, 6) or (6, 1).
